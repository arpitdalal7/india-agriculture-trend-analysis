# Agricultural Production Trend Analysis in India 🌾📊

This project analyzes **agricultural production trends in India** using an interactive BI dashboard. It focuses on crop-wise and state-wise patterns across food grains, oilseeds, pulses, and commercial crops.

---

## 🌍 Project Overview

- Places Indian agriculture in the **global economic context**  
- Shows how agriculture fits within **India’s GDP and employment**  
- Highlights **land use, crop mix, and spatial distribution** of production  
- Uses visuals to compare **major crops and leading states**

---

## 🔎 Key Focus Areas

- **Oilseeds:** Groundnut, soybean, rapeseed & mustard, and others  
- **Food grains:** Rice, wheat, maize, millets, etc.  
- **Pulses:** Gram, tur, urad, moong, lentil, and others  
- **Commercial crops:** Sugarcane, cotton, jute & mesta, etc.  
- **Spatial distribution:** Which states lead in which crop groups

> The attached BI dashboard PDF provides all charts and detailed insights.
